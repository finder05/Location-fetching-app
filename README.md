# 📍 Location Fetch Android App

## 📌 Overview
This Android application fetches the current device location (Latitude and Longitude) and displays it using an Alert Dialog.

---

## 🎯 Features
- Fetch current location using GPS
- Displays Latitude and Longitude
- Runtime permission handling
- Simple and user-friendly interface

--


## 🛠️ Technologies Used
- Java
- Android Studio
- Google Play Services Location API

---

## 📱 Requirements
- Android Studio (latest version)
- Android device or emulator
- Minimum SDK: API 21+
  
---

## 🔐 Permissions Used
```xml

<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION"/>
<uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION"/>


```
